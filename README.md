# How to
为了能够提交文章，首先是需要把代码 fork 下来，能在本地运行后，把新的文章提交上来。再提交Pull Request，得到合并以后就能在网站看到你提交的博文了。

当你能成功提交并发布一片文章，你就学会了如何构建一个个人文档或者博客网站，并且懂得了 Github 如何提 PR 了。

最后随时欢迎大家提交 pr
